Hola,para empezar a utilizar el módulo debes dirigirte al editor de VBA, una vez dentro activa la referencia
Microsft XML v6.0.

Para generar un archivo de imagen hacemos uso del siguiente recurso:

https://chart.googleapis.com/chart?chs=150x150&cht=qr&chl=mi_contenido

la palabra 'mi_contenido' puede ser reemplazado por valor que desees para generar tu QR,puedes copiar y pegarlo
en tu navegador y te mostrará el código QR.

Bien para llevar del navegador a un archivo de imagen manipulable.Procedemos a recuperar la respuesta del servidor en un array de bytes y lo escribimos en un archivo en la ruta que deseamos.

Y eso es todo...,ahora puedes usar la imagen.
